# Drunk Cards 🃏

A local multiplayer party card game built as a PWA. Players take turns drawing cards and completing challenges.

## Features

- Card draw mechanics with challenge-based gameplay
- Smooth animations and audio feedback
- Works offline as a Progressive Web App (PWA)
- Mobile-first, no backend required

## Tech Stack

React · TypeScript · styled-components · Framer Motion · Howler.js · Zustand · Vite · PWA

## Run locally

npm install
npm run dev
